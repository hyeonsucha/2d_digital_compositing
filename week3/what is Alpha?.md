# What is Alpha?

디지털 이미지에서 채널이란 전체 픽셀 정보를 구성하는 개별 색상 값에 대한 정보를 말한다. 예를 들어 RGB이미지에는 픽셀의 빨강, 녹색, 파랑 값을 개별적으로 정의하는 세 개의 채널이 존재한다.

이미지 소프트웨어에서는 각각의 채널 속 단색 말고도 직접 표시되지 않는 색의 데이터를 넣을 수 있다. 이것을 알파채널이라 하며 투명, 반투명 효과에 사용한다.
